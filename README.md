# 📋 Daily Task Tracker Bot

## Деплой на Railway (бесплатно)

### 1. Получи токен бота
- Открой Telegram → @BotFather → `/newbot`
- Скопируй токен вида `123456:ABC-DEF...`

### 2. Загрузи код на GitHub
- Зайди на github.com → New repository → назови `task-bot`
- Загрузи все файлы из этой папки

### 3. Задеплой на Railway
1. Зайди на **railway.app** (вход через GitHub)
2. **New Project → Deploy from GitHub repo** → выбери `task-bot`
3. Нажми на проект → **Variables** → добавь:
   - `BOT_TOKEN` = твой токен от BotFather
4. Railway автоматически запустит бота ✅

### Команды бота
- `/start` — открыть трекер
- Нажимай на задачи чтобы отмечать выполнение
- `➕ Добавить задачу` — добавить свою задачу
- `📊 История` — статистика за 7 дней

### Формат добавления задачи
```
🏋️ Тренировка : 25
📝 Дневник : 10
```
